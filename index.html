Projects

Stand out project
Made in the Java Processing framework. This project is a fully animated interactive 2D scene of a car driving along a road. In it, the tracks on the road, the cars wheels and the background are all animated in tandem with each other to create the illusion that the car is actually travelling down the road. Likewise, the user is able to control the position of the car with their cursor.
An interactive arcade style 2D game made in the Java processing framework. Players use the keyboard to move around the red catcher at the bottom of the screen. The objective is to catch as many of the blue balls that spawn at random from out of the sky as possible before they fall down off the edge of the screen. Players are rewarded points for catching balls with the number of points won in correlation to the size of the ball they caught. The game ends when the player fails to catch a ball.
A cool racing simulation made in the Java processing framework. Users are presented with 6 race cars and watch them as they speed towards the finish line. Because each car moves at random speeds, no race is pre-determined. Each race can be expected to play out differently from each other. The first car to reach the finish line will win a point. After the race has ended, users can start a new race. Cars scores are preserved on the beginning of new races.
